## 小组成员及分工

### 组长 — 框架核心与项目整合
| 负责内容 | 对应文件 |
|---------|---------|
| 游戏引擎核心类扩展 | ElementObj.java |
| 元素管理器与枚举扩展 | ElementManager.java, GameElement.java |
| 配置文件加载器 | GameLoad.java |
| 游戏主循环与碰撞逻辑 | GameThread.java |
| 键盘监听控制 | GameListener.java |
| 渲染引擎/摄像机会/hud | GameMainJPanel.java, GameJFrame.java |
| 中文UI面板 | ChineseGamePanel.java |
| 增强键盘输入 | GameInputDispatcher.java |
| 游戏入口 | GameStart.java |
| 配置文件 | obj.pro, GameData.pro |

### 组员A — 关卡与场景系统
| 负责内容 | 对应文件 |
|---------|---------|
| 地形绘制（草地/石台） | Platform.java |
| 可破坏障碍物（木箱/油桶） | Barrier.java |
| 背景标记 | Background.java |
| 掉落道具（武器/回血/加命） | ItemDrop.java |
| 爆炸特效（粒子动画） | Explosion.java |
| 关卡地图配置 | lev1.map, lev2.map, lev3.map |

### 组员B — 角色与敌人系统
| 负责内容 | 对应文件 |
|---------|---------|
| 玩家控制（移动/跳跃/射击/武器） | Player.java |
| 子弹系统（4种武器弹道） | Bullet.java |
| 敌人AI（巡逻/冲锋/狙击） | Enemy.java |
| Boss（双形态攻击） | Boss.java |
